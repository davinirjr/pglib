Welcome to pglib
================

pglib is a Python 3.3+ module for working with PostgreSQL databases.  It is a C extension
that exposes the `libpq <http://www.postgresql.org/docs/9.3/static/libpq.html>`_ API.
It is designed to be small, fast, and as convenient as possible.
